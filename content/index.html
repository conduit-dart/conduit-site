One framework, all the tools you need.

Fluid routing

A familiar, higher-order functional syntax makes application flow easy to construct and read.

{% code dart %} router .route('/teams/[:id]') .link(() => Authorizer(validator)) .link(() => TeamController(databaseContext)); {% endcode %}

Statically-typed ORM with database migrations

Powerful, statically typed queries enable code-completion and refactoring tools while minimizing error. Tested command-line tooling for database migrations makes data management a breeze.

{% code dart %} final query = Query(context) ..where((t) => t.skills).contains('conduit') ..join(object: (t) => t.manager) ..join(set: (t) => t.developers); final teams = await query.fetch(); {% endcode %}



Works with your favorite technology

Conduit is built on the word class Dart platform, which powers some of the world’s most trusted apps and frameworks, such as Google Ads and Flutter. Dart’s maturity and active developer community means Conduit works with all of the most popular technology and services.

And with prebuilt Docker images and built-in support for PostgreSQL and MySQL databases, Conduit is ready for your tech stack.


Reliable, fast, and tested

Conduit is relied upon to power advanced applications in production for several companies, including TaxiTecnic. Built-in concurrency means it’s blazing fast, and memory-isolated threads eliminate complicated synchronization and race condition worries. Conduit is supported by over 1400 automated tests, guaranteeing the stability of each release. And features like sound null safety allow for confidence when writing an application.
